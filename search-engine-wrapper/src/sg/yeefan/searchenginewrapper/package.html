Provides a wrapper framework for unifying programatic access to search engines.

The main class for this package is {@link sg.yeefan.searchenginewrapper.SearchEngineWrapper SearchEngineWrapper}, to be used in conjunction with one of the search engine clients subclassed from {@link sg.yeefan.searchenginewrapper.SearchEngineClient SearchEngineClient} or {@link sg.yeefan.searchenginewrapper.KeyedSearchEngineClient KeyedSearchEngineClient}. @author Tan Yee Fan
