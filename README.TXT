INTRODUCTION
------------

This repository contains three projects:
1. Search Engine Wrapper (main project)
2. Charset Detect Stream Reader (supporting project for Search Engine Wrapper)
3. File Downloader (supporting project for Search Engine Wrapper)

All files for each of the above projects is contained in its own subdirectory.
Such files include source code, required components, as well as documentation.
Each project is built separately.
